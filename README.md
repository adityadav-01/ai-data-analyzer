# AI Data Analyzer Web Application

AI-powered web application that performs automated data cleaning, exploratory data analysis (EDA), visualization, time series forecasting, and machine learning prediction.

## Features
- CSV / Excel Upload
- Automatic Missing Value Handling
- Statistical Summary
- Interactive Visualizations
- Correlation Heatmap
- Time Series Forecasting (Prophet)
- Regression Modeling (Random Forest)
- Download Cleaned Data
- Download Forecast Results

## Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Seaborn
- Prophet
- Scikit-learn
